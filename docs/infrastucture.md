## Infrastructure Description

The Udagram application is a Full-Stack web application that demos the core components of a typical web application.

We are using the following infrastructure:

-   Backend build with Node.js & TypeScript.
-   Postgres database.
-   Front-End build using Angular.js.
-   GitHub & Gitlab are used to host the source code.
-   Circle CI handles the CI/CD pipelines.

### Hosting

-   AWS RDS - Postgres Database.
-   AWS Beanstalk - web server environment.
-   AWS S3 - static front-end web application.
